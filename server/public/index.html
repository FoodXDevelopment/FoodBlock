Quick Examples

Trace Provenance Chain

Enter a block hash to walk its update chain back to the genesis block. 

FoodBlock {
  type    what it is
  state   its properties
  refs    what it references
}

id = SHA-256(type + state + refs)

Endpoints

Base URL: /foodblock

GET /foodblock/health Health check
POST /foodblock/blocks Create a block
GET /foodblock/blocks/:hash Get block by hash
GET /foodblock/blocks?type=...&heads=true Query blocks
GET /foodblock/chain/:hash Provenance chain
GET /foodblock/heads All head blocks

Quick Start

terminal
# Create a product block curl -X POST https://api.foodx.world/foodblock/blocks \ -H "Content-Type: application/json" \ -d '{ "type": "substance.product", "state": {"name": "Sourdough Loaf", "organic": true}, "refs": {} }' # Query all substance blocks curl https://api.foodx.world/foodblock/blocks?type=substance # Get a specific block curl https://api.foodx.world/foodblock/blocks/{hash} # Trace provenance chain curl https://api.foodx.world/foodblock/chain/{hash}
